# Smart Student Database System (JDBC + MySQL)

## Project Overview

The **Smart Student Database System** is a console-based Java application developed using **Core Java**, **JDBC**, and **MySQL**. The application performs CRUD (Create, Read, Update, Delete) operations by connecting to a MySQL database, providing persistent storage for student records.

---

## Features

* Add Student
* View All Students
* Search Student by ID
* Update Student Details
* Delete Student Record
* MySQL Database Integration
* JDBC Connectivity
* Exception Handling
* Menu-Driven Console Application
